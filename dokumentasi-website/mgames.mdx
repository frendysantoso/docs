MGames dilengkapi dengan fitur pembelian Guest, yang dimana pengguna atau user dapat bertransaksi tanpa harus login. MGames juga dilengkapi sistem pembayaran menggunakan saldo akun.
MGames dibuat menggunakan Framework CodeIgniter 4 dan dikembangkan oleh Enivay Company. MGames dilengkapi dengan sistem otomatis 2 Provider Transaksi dan 1 Provider Payment Gateway.'

### Membuat Merchant di Tripay

Untuk membuat merchant atau menambah merchant di Tripay, kamu dapat menggunakan cara berikut :

1. Login tripay menggunakan akun kamu.
2. Masuk ke menu "Merchant".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1724311675/tri_npkhks.png" />
3. Klik "Tambah Merchant".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1724311676/tri1_qiflvq.png" />

    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1724311676/tri2_kyub5a.png" />
    - Nama Merchant : Isi dengan nama merchant atau toko kamu.
    - Logo : Isi dengan logo merchant atau toko
    - URL Website : Isi dengan URL atau link website
    - Deksripsi : Isi dengan penjelasan atau deksripsi singkat tentang bisnis atau toko kamu
    - URL Callback : Isi dengan URL Callback yang tertera di kolom Konfigurasi Tripay, atau klik link https://demo.enivay.com/mgames/admin/konfigurasi
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1724311675/tri3_c2hbaq.pngg" />
    - Whitelist IP : Isi dengan IPv6 dari server hosting kamu
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1724311676/tri4_rrviw2.png" />
