## Admin

### Login Admin

Berikut cara login admin :

1. Silahkan tambahkan “/admin” pada ujung link website.
2. Masukkan “Username” & “Password” akun admin kamu.
  <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732087991/Login_Admin_mqbdpg.png" />
    - Masukkan username akun kamu.
    - Isi dengan password akun kamu.
3. Jika sudah klik "Login Sekarang”.
  <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732087991/Frame_439_biamj2.png" />


### Konfigurasi Umum

Tab Umum pada menu Konfigurasi merupakan menu untuk melakukan konfigurasi seperti mengganti Judul Website, Judul Halaman Utama, Icon Website, Logo, Author, Keywords & Deskripsi.

Berikut langkah untuk melakukan Konfigurasi Umum :

1. Pada halaman dashboard admin, silahkan klik menu “Konfigurasi” pada menu utama seperti berikut.
  <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732088130/Konfigurasi_e62jsx.png" />
2. Edit kolom Konfigurasi Umum sebagai berikut :
  <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732088123/Frame_441_pvrb8k.png" />
    - Judul Website : Isi dengan judul website kamu.
    - Judul Halaman Utama : Isi dengan judul yang kamu inginkan.
    - Icon : Isi dengan icon website kamu dengan ukuran 124 x 124 px.
    - Logo : Pilih atau masukkan logo website kamu dengan ukuran 160 x25 px.
    - Author : Isi dengan nama kamu.
    - Keywords : Isi dengan keywords atau kata kunci yang sesuai dengan website kamu.
    - Deksripsi : Isi dengan deksripsi singkat tentang website kamu.
3. Jika sudah klik “Submit”.
  <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732088125/Frame_442_i9oubz.png" />


### Konfigurasi Tripay

Tab Tripay pada menu Konfigurasi merupakan menu untuk mengelola sistem pembayaran website kamu dengan sistem Tripay.
Berikut langkah untuk Konfigurasi Tripay :

1. Pada halaman dashboard admin, silahkan klik menu “Konfigurasi” pada menu utama seperti berikut.
  <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732088130/Konfigurasi_e62jsx.png" />
2. Silahkan edit kolom konfigurasi yang diinginkan dengan cara memasukkan “Api Key”, “Private Key” & “Kode Merchant” kamu.
  <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732088125/Frame_443_ptmj1w.png" />
3. Jika sudah klik “Submit”.
  <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732088126/Frame_444_kzxim5.png" />

**Note !!**

Untuk mendapatkan api key, private key, dan kode merchant kamu bisa klik "Edit" pada merchant kamu. Kemudian salin "Api Key", Private Key", & "Kode Merchant" berikut.
<img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1721458238/tri_kwxj8s.png" />

Menu Kelola Metode merupakan menu untuk menampilkan seluruh data metode pembayaran di website kamu. Pada menu ini kamu dapat melakukan tambah metode, edit metode, & hapus metode.

Berikut langkah untuk mengelola metode :

<Accordion title="Tambah Metode">

        *Transfer Bank (BCA, BRI, BNI, Mandiri, dll.)
