### Konfigurasi Umum

Tab Umum pada menu Konfigurasi merupakan menu untuk mengganti Nama Website, Icon Website, Logo Website, Author, Keywords, & Deksripsi. Berikut langkah untuk melakukan konfigurasi Umum : 

1. Klik "Konfigurasi" pada halaman admin.
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732938687/Konfigurasi_xs4oju.png" />
2. Klik tab Umum.
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732938666/Frame_868_pjpwas.png" />
3. Silahkan edit atau ganti konfigurasi yang diinginkan sebagai berikut : 
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732938667/Frame_869_tkninf.png" />
    - Nama Website : Isi dengan Nama Website kamu.
    - Icon Website : Isi dengan link icon website kamu.
    - Logo Website : Isi dengan link logo website kamu.
    - Author : Isi dengan nama author website kamu.
    - Keywords : Isi sesuai dengan nama website kamu.
    - Deksripsi : Isi dengan deksripsi singkat website kamu.
    - Pembayaran Saldo : Pilih "On" jika menggunakan sistem pembayaran saldo. Pilih "Off" jika tidak menggunakan sistem pembayaran saldo.
    - Lisensi Validasi Games : Silahkan hubungi developer untuk mendapatkan kode lisensi. 
4. Jika sudah klik "Simpan".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732938668/Frame_870_ggb4an.png" />

### Membuat Merchant di Tripay

Untuk membuat merchant atau menambah merchant di Tripay, kamu dapat menggunakan cara berikut : 

1. Login tripay menggunakan akun kamu.
2. Masuk ke menu "Merchant".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1723267372/mt_glrdc5.png" />
3. Klik "Tambah Merchant".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1723267371/mt1_bjqze9.png" />
4. Isi kelengkapan data sebagai berikut : 
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1723267370/mt2_hhqx4u.png" />
    - Nama Merchant : Isi dengan nama merchant atau toko kamu.
    - Logo : Isi dengan logo merchant atau toko
    - URL Website : Isi dengan URL atau link website
    - Deksripsi : Isi dengan penjelasan atau deksripsi singkat tentang bisnis atau toko kamu
    - URL Callback : Isi dengan URL Callback yang tertera di kolom Konfigurasi Tripay, atau klik link https://demo.enivay.com/vigames/admin/config dibagian tab Tripay 
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1723267369/mt3_sql8nx.png" />
    - Whitelist IP : Isi dengan IPv6 dari server hosting kamu
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1723267368/mt4_tglrtd.png" />
5. Jika sudah klik "Simpan".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1723267367/mt5_lywn7u.png" />

Kelola Games merupakan menu untuk melihat seluruh games yang telah ditambahkan dan ditampilkan ke website kamu. Selain itu kamu juga dapat melakukan tambah atau edit games di menu ini.

<Accordion title="Tambah Games">
1. Klik menu "Kelola Games" pada halaman admin.
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732939898/Kelola_Games_sz6efs.png" />

    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732939888/Frame_900_sjnj6d.png" />

    - Nama Game : Isi dengan nama games yang ingin ditambahkan.
    - Slug : Isi dengan nama games dengan huruf kecil. (Contoh : Mobile Legend -> mobile-legend).
    - Kategori : Pilih salah satu kategori yang telah ditambahkan sebelumnya.
    - Deskripsi : Isi dengan deksripsi singkat tentang games yang ditambahkan.

<Accordion title="Edit Topup">
1. Klik menu "Kelola TopUp" pada halaman admin.

### Kelola Pembelian

Kelola Pembelian merupakan menu untuk melihat seluruh data pembelian dari seluruh member di website kamu. Pada menu Kelola Pembelian kamu dapat melakukan edit data pembelian.

Berikut cara Edit Pembelian : 

1. Klik menu "Kelola Pembelian" pada halaman admin.
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1732941753/Kelola_Pembelian_athlyd.png" />
