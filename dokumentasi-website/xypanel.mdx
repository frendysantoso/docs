### Daftar Akun

Cara daftar akun dapat dilakukan dengan cara berikut : 

1. Pada tampilan awal atau dashboard silahkan klik "Registrasi".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1731907838/Daftar_Akun_s1wyaa.png" />
2. Lengkapi data sebagai berikut.
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1731907838/Frame_129_gdq4me.png" />

### Kelola Provider

Kelola Provider merupakan menu yang menampilkan dan mengelola seluruh data provider yang terpasang di website kamu. 

Pada menu Kelola Provider kamu dapat melakukan Tambah Provider, Edit Provider, Hapus Provider, & Konfigurasi Provider.

Berikut cara untuk Kelola Provider : 

<Accordion title="Tambah Provider.">
1. Pada tampilan awal atau dashboard setelah login silahkan klik menu "Provider".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1731914592/Kelola_provider_klna1q.png" />
2. Klik "Tambah Provider" pada kolom Data Provider.
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1731914594/Tambah_Provider_hgouxd.png" />
3. Lengkapi data sebagai berikut.
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1731914582/Frame_172_jj1dt5.png" />
    - Nama Provider : Isi dengan nama provider yang ingin ditambahkan.
      Contoh : Borneo Panel - SMM
    - Link : Masukkan link api website provider yang ingin ditambahkan. Contoh : https://borneopanel.com/api/

      **Note :**
      Untuk mendapatkan link api provider kamu dapat masuk ke menu "Api" atau "Dokumentasi Api" pada website provider, kemudian salin "API Key" atau "API URL" yang tersedia.
        <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1723446475/pro.note1_r7y68t.png" />

    - Api Key : Masukkan Api Key website provider. Contoh : uAgiidsI2bdz1BKUzjcRwxn3

      **Note :**
      Untuk mendapatkan api key provider kamu dapat masuk ke menu "akun" atau profile" pada website provider, kemudian salin "URL Request" atau "API URL" yang tersedia.
        <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1723446474/pro.note2_xaykq9.png" />

    - Tipe : Pilih tipe berupa : 
      Luar -> Jika web provider dari luar negeri
      Indo -> Jika web provider dari indonesia.

      On -> Jika ingin menghidupkan atau mengaktifkan provider ke web kamu.
      Off -> Jika ingin mematikan atau menon-aktifkan provider ke web kamu.

4. Jika sudah klik "Simpan".

### Kelola Pengguna

Menu "Pengguna" merupakan menu yang digunakan untuk menampilkan dan mengelola seluruh data pengguna website kamu. 

Pada menu ini kamu dapat melakukan import data pengguna jika sebelumnya sudah memiliki data pengguna, kamu bisa import data pengguna kamu dengan menggunakan fitur import. Selain itu kamu juga dapat melakukan edit serta hapus data pengguna.

Berikut langkah untuk mengelola data pengguna : 

<Accordion title="Edit Pengguna">
1. Pada menu admin silahkan klik menu "Pengguna".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1731916064/Kelola_Pengguna_xzitd8.png" />
2. Klik "Edit" pada data pengguna yang ingin di edit.
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1731916060/Frame_180_omtbmr.png" />
3. Edit data pengguna sebagai berikut : 
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1731916057/Frame_178_kjdvbf.png" />
    - Nama Lengkap : Silahkan isi dengan nama lengkap pengguna.
    - Username : Default (tidak dapat diganti/diedit).
    - Email : Silahkan isi dengan alamat email pengguna.
    - Saldo : Isi dengan jumlah saldo pengguna.
    - Password : Silahkan klik "Reset Password" jika pengguna mengalami lupa password.
    - Status : Pilih "On" jika pengguna masih aktif, pilih "Off" jika pengguna sudah tidak aktif.
    - Level : Pilih "Member" jika level pengguna adalah member. Pilih "Admin" jika level pengguna adalah admin
    - Api Key : Klik "Generate Ulang" jika ingin mengganti atau melakukan generate pada api key pengguna.
    - Mode : Pilih mode "Light" jika ingin menggunakan mode terang. Pilih mode "Dark" jika ingin menggunakan mode gelap.
    - Terdaftar : Default (tidak dapat diganti/diedit).
4. Jika sudah, klik "Simpan".
