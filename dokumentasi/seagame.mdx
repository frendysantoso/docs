Berikut langkah untuk melakukan Konfigurasi Umum :

1. Pada halaman dashboard admin, silahkan klik menu "Konfigurasi" pada menu utama seperti berikut.
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1731305810/Konfigurasi1_ll0cqf.png" />
2. Klik tab umum.
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1731306490/Konfigurasi2_awvqya.png" />
3. Edit kolom konfigurasi sebagai berikut :
    <img width="50%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1731305826/Konfigurasi_Umum_taq7de.png" />
    - Nama Website : Isi dengan nama website kamu.
    - Judul Halaman Utama : Isi dengan judul yang kamu inginkan.
    - Icon : Isi dengan icon website kamu dengan ukuran 256 x 256 px.
    - Logo Website : Pilih atau masukkan logo website kamu.
    - Author : Isi dengan nama website kamu.
    - Keywords : Isi dengan keywords atau kata kunci yang sesuai dengan website kamu.
    - Deksripsi : Isi dengan deksripsi singkat tentang website kamu.
