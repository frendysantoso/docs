#### Login Admin

Berikut cara untuk login sebagai admin :

1. Silahkan tambahkan "/admin" pada ujung link website seperti berikut.
2. Masukkan "Username" & "Password" akun admin kamu
    - Username : Masukkan username akun kamu
    - Password : Isi dengan password akun kamu
3. Jika sudah klik "Masuk Sekarang".

      Contoh : Halo *#username#*
      Pembelian kamu dengan ID *#order_id#* telah berhasil sukses.

#### Daftar Member

Untuk login sebagai member dapat dilakukan dengan cara berikut :

1. Pada halaman utama klik "Daftar"
2. Silahkan registrasi akun kamu sebagai berikut :
    - Username :Silahkan isi dengan username yang kamu inginkan.
    - No. Whatsapp : Isi dengan nomor whatsapp kamu.
    - Password : Isi dengan password kamu.
    - Ulangi Password.
3. Jika sudah, klik "Submit".


#### Login Sebagai Member

Untuk login sebagai member dapat dilakukan dengan cara berikut :

1. Pada halaman utama klik "Masuk".
2. Masukkan Username & Password kamu.
3. Kemudian klik "Masuk Sekarang"


#### Lupa Password

Untuk mengatasi lupa password, member dapat melakukan langkah sebagai berikut :

1. Pada halaman utama klik "Masuk".
2. Pada kolom login, silahkan klik "Lupa Password" seperti berikut.
3. Silahkan masukkan username kamu.
4. Jika sudah klik "Reset Password"


#### Transaksi Member

Transaksi Member merupakan transaksi yang dapat dilakukan oleh pengguna atau member setelah melakukan login.

Berikut langkah untuk transaksi member :

1. Pada halaman utama, silahkan pilih kategori games yang ingin di TopUp. Contoh : Kategori Games Mobile.
2. Klik salah satu game yang ingin kamu Top Up. Contoh : Mobile Legend.
3. Silahkan lengkapi data kamu sebagai berikut.
    - Masukkan ID kamu.
    - Pilih Nominal Top Up.
    - Pilih metode Pembayaran.
    - Masukkan Nomor WhatsApp kamu jika kamu ingin mendapatkan bukti pembayaran atas pembelian yang kamu lakukan.
4. Jika sudah klik "Beli Sekarang".
5. Silahkan klik "Konfirmasi" pada popup pesanan tersebut jika ingin mengkonfirmasi pesanan kamu.
6. Jika sudah, maka akan muncul detail pesanan seperti berikut.
7. Langkah selanjutnya, silahkan tranfer ke rekening yang tertera sesuai dengan jumlah total pembayaran. Pesanan mu akan segera di proses.


#### Cek Pesanan Member

Berikut langkah untuk mengecek pesanan atau riwayat pembelian kamu :

1. Pada menu halaman utama silahkan klik menu “Cek Pesanan”.
2. Jika sudah, maka tampilan menu cek pesanan akan seperti berikut.
3. Jika ingin melihat detail pesanan, kamu bisa klik tanda panah pada pesanan yang ingin kamu lihat detailnya.
4. Maka akan muncul tampilan detail pesanan seperti berikut.


#### Topup Saldo

Untuk melakukan topup saldo, kamu dapat melakukan cara berikut :

1. Pada menu halaman utama silahkan klik foto profile/user pada pojok kanan atas seperti berikut :
2. Kemudian silahkan klik menu “TopUp” seperti berikut.
3. Selanjutnya silahkan masukkan nominal topup.
4. Pilih metode pembayaran.
5. Jika sudah, silahkan klik “Top Up”.


#### Ganti Password

Untuk mengganti password, kamu dapat melakukan cara berikut :

1. Pada menu halaman utama silahkan klik menu “Beranda”.
2. Jika sudah silahkan scroll ke kolom ganti password seperti berikut.
3. Silahkan masukkan Password baru kamu & Konfirmasi password kamu.
4. Jika sudah, silahkan klik “Simpan”.
