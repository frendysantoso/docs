#### Login Admin

Berikut cara untuk login sebagai admin : 

1. Silahkan tambahkan "/admin" pada ujung link website seperti berikut. 
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1721378140/login1_jkyuxb.png" />
2. Masukkan "Username" & "Password" akun admin kamu.
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1721456148/LA1_ofrglh.png" />
    - Username : Masukkan username akun kamu.
    - Password : Isi dengan password akun kamu.
3. Jika sudah klik "Log In".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1721456147/LA2_offxfr.png" />



#### Konfigurasi Umum


Tab Umum pada menu Konfigurasi merupakan menu untuk melakukan konfigurasi seperti mengganti Icon Website, Logo (White & Color), Judul Website, Nama Website, Author, Keywords, Deskripsi, Pembayaran Sal

Berikut langkah untuk melakukan Konfigurasi Umum : 

1. Klik "Konfigurasi" pada menu admin.
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1721456253/KU1_szjv5v.png" />
2. Klik tab "Umum".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1721456252/KU2_xysgdw.png" />
3. Silahkan edit kolom konfigurasi sebagai berikut : 
    <img width="30%" src=https://res.cloudinary.com/dvnlxmymx/image/upload/v1721456252/KU3_ilvrgg.png" />
    - Icon : Isi dengan icon website kamu dengan ukuran 512 x 512 px 
    - Logo Website 
        * Logo White : Pilih atau masukkan logo putih website kamu.
        * Logo Color : Pilih atau masukkan logo berwarna website kamu.
    - Judul : Isi dengan judul webiste kamu.
    - Nama Web : Isi dengan nama website kamu.
    - Author : Isi dengan nama website kamu.
    - Keywords : Isi dengan keywords atau kata kunci yang sesuai dengan website kamu.
    - Deksripsi : Isi dengan deksripsi singkat tentang website kamu.
    - Pembayaran Saldo : Pilih "Ya", jika website kamu ingin menggunakan pembayaran via saldo. Pilih "Tidak" jika website kamu tidak ingin menggunakan pembayaran via saldo.
    - License Api Enivay : Lisensi ini digunakan untuk melakukan Validasi ID player games. Untuk mendapatkan lisensi bisa hubungi +62 856 5400 8642.
4. Jika sudah klik "Simpan".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1721456289/simpan_uwghgb.png" />
