#### Daftar Akun

Cara daftar akun dapat dilakukan dengan cara berikut : 

1. Pada tampilan awal atau dashboard silahkan klik "Registrasi".
2. Lengkapi data sebagai berikut.
    - Nama Lengkap : Isi dengan nama lengkap kamu.
    - Username : Masukkan username akun yang kamu inginkan.
    - Email : Masukkan email kamu yang masih aktif (valid).
    - Password : Isi dengan password yang kamu inginkan.
    - Klik kolom centang "Saya setuju dengan syarat & ketentuan" yang menyatakan kamu menyetujui semua syarat & ketentuan yang telah ditetapkan oleh XY Panel.
3. Jika sudah klik "Register".
4. Jika akun sudah berhasil dibuat maka akan muncul pemberitahuan cek email untuk verifikasi akun kamu.


#### Login

Untuk login sebagai admin maupun member dapat dilakukan dengan cara berikut : 

1. Pada tampilan awal atau dashboard silahkan klik "Login".
2. Masukkan Username & Password akun kamu.
    - Username : Masukkan username akun kamu.
    - Password : Isi dengan password akun kamu.
    - Klik kolom centang "Remember Me" jika ingin tetap login selama 30 hari.
3. Jika sudah klik "Login".


#### Lupa Password

Untuk login sebagai admin maupun member dapat dilakukan dengan cara berikut : 

1. Pada tampilan awal atau dashboard silahkan klik "Login".
2. Klik "Lupa Password?".
3. Masukkan username kamu
4. Jika sudah klik "Reset Password".


#### Konfigurasi Umum

Tab Umum pada menu Konfigurasi merupakan menu untuk melakukan konfigurasi seperti mengganti Nama Website, Icon Website, Author, Keywords, & Description. 

Berikut langkah untuk melakukan Konfigurasi Umum : 

1. Pada menu admin silahkan klik menu "Konfigurasi".
2. Klik tab "Umum"
3. Lengkapi data sebagai berikut.
    - Nama Website : Isi dengan nama website kamu.
    - Icon : Masukkan icon website kamu.
    - Author : Masukkan nama author.
    - Keywords : Isi dengan keywords yang sesuai dengan website kamu.
    - Description : Isi dengan deksripsi singkat website kamu.
4. Jika sudah klik "Simpan".
5. Jika sudah berhasil disimpan maka akan muncul pemberitahuan seperti berikut.


#### Konfigurasi Tripay

Tab Tripay pada menu Konfigurasi merupakan menu untuk melakukan konfigurasi pembayaran website kamu dengan sistem tripay. 

Berikut langkah untuk Konfigurasi Tripay : 

1. Pada menu admin silahkan klik menu "Konfigurasi".
2. Klik tab "Tripay".
3. Silahkan masukkan Api Key, Private Key, & Kode Merchant toko kamu ke kolom "Api Key", Private Key", & "Kode Merchant". 
4. Jika sudah klik "Simpan".

***Note :**
Untuk mendapatkan api key, private key, dan kode merchant kamu bisa klik "Edit" pada merchant kamu. Kemudian salin "Api Key", Private Key", & "Kode Merchant" berikut.


#### Konfigurasi Telegram

Tab Telegram pada menu Konfigurasi merupakan menu untuk melakukan konfigurasi agar kamu dapat mengirim notifikasi kepada member website kamu. 

Berikut langkah untuk melakukan Konfigurasi Telegram : 

1. Pada menu admin silahkan klik menu "Konfigurasi".
2. Klik tab "Telegram"
3. Lengkapi data sebagai berikut.
    - Chat ID : Untuk mendapatkan chat id silahkan start bot berikut @get_id_bot
    - Notifikasi : Pilin "On" untu menghidupkan notifikasi, pilih "Off" jika ingin mematikan notifikasi.
4. Jika sudah klik "Simpan".
5. Jika sudah berhasil disimpan maka akan muncul pemberitahuan seperti berikut.

#### Konfigurasi SMTP

Tab SMTP pada menu Konfigurasi merupakan menu untuk melakukan konfigurasi agar kamu dapat mengirimkan pesan email kepada member website kamu. 

Berikut langkah untuk melakukan Konfigurasi SMTP : 

1. Pada menu admin silahkan klik menu "Konfigurasi".
2. Klik tab "SMTP".
3. Lengkapi data sebagai berikut.
    - Status : Pilih "On" jika ingin menghidupkan konfigurasi SMTP, Pilih "Off" jika tidak ingin menghidupkan konfigurasi SMTP.
    - Host : Pilin on untuk menghidupkan notifikasi, pilih off jika ingin mematikan notifikasi.
    - Email : Pilin on untuk menghidupkan notifikasi, pilih off jika ingin mematikan notifikasi.
    - Password : Pilin on untuk menghidupkan notifikasi, pilih off jika ingin mematikan notifikasi.
    - Port : Pilin on untuk menghidupkan notifikasi, pilih off jika ingin mematikan notifikasi.
4. Jika sudah klik "Simpan".
5. Jika sudah berhasil disimpan maka akan muncul pemberitahuan seperti berikut.


#### Top 10 Pembelian

Berikut langkah untuk mengelola Top 10 Pembelian :

  </Tab>
  <Tab title="Edit Top 10 Pembelian">
1. Pada menu admin silahkan klik menu "Top 10".
2. Klik "Edit" pada kolom Top 10 Pembelian yang ingin di edit
3. Edit data sebagai berikut.
    - Username : Default, tidak dapat di edit.
    - Jumlah : Masukkan total jumlah pembelian.
    - Nominal : Masukkan nominal pembelian.
    - Periode : Pilih periode pembelian yang kamu inginkan.
4. Jika sudah klik "Simpan".
5. Jika sudah berhasil disimpan maka akan muncul pemberitahuan seperti berikut.
  </Tab>
  <Tab title="Hapus Top 10 Pembelian">

#### Kelola Provider

Kelola Provider merupakan menu yang menampilkan dan mengelola seluruh data provider yang terpasang di website kamu. 

Pada menu Kelola Provider kamu dapat melakukan Tambah Provider, Edit Provider, Hapus Provider, & Konfigurasi Provider.

Berikut cara untuk Kelola Provider : 

<Tabs>
  <Tab title="Tambah Provider">
1. Pada tampilan awal atau dashboard setelah login silahkan klik menu "Top 10".
2. Klik "Tambah Bot" pada kolom Top 10 Pembelian.
3. Lengkapi data sebagai berikut.
    - Nama Provider : Isi dengan nama provider yang ingin ditambahkan.
      Contoh : Borneo Panel - SMM
    - Link : Masukkan link api website provider yang ingin ditambahkan. Contoh : https://borneopanel.com/api/

      **Note :**
      Untuk mendapatkan link api provider kamu dapat masuk ke menu "Api" atau "Dokumentasi Api" pada website provider, kemudian salin "API Key" atau "API URL" yang tersedia.

    - Api Key : Masukkan Api Key website provider. Contoh : uAgiidsI2bdz1BKUzjcRwxn3

      **Note :**
      Untuk mendapatkan api key provider kamu dapat masuk ke menu "akun" atau profile" pada website provider, kemudian salin "URL Request" atau "API URL" yang tersedia.

    - Tipe : Pilih tipe berupa : 
      Luar -> Jika web provider dari luar negeri
      Indo -> Jika web provider dari indonesia.
    - Status : Pilih status berupa : 
      On -> Jika ingin menghidupkan atau mengaktifkan provider ke web kamu.
      Off -> Jika ingin mematikan atau menon-aktifkan provider ke web kamu.

4. Jika sudah klik "Simpan".
5. Jika sudah berhasil disimpan maka akan muncul pemberitahuan seperti berikut.
  </Tab>
  <Tab title="Edit Provider">
1. Pada tampilan awal atau dashboard setelah login silahkan klik menu "Top 10".
2. Klik "Edit" pada kolom Top 10 Pembelian yang ingin di edit
3. Edit data sebagai berikut.
    - Nama Provider : Isi dengan nama provider yang ingin ditambahkan.
      Contoh : Borneo Panel - SMM
    - Link : Masukkan link api website provider yang ingin ditambahkan. Contoh : https://borneopanel.com/api/
      
      **Note :**
      Untuk mendapatkan link api provider kamu dapat masuk ke menu "Api" atau "Dokumentasi Api" pada website provider, kemudian salin "API Key" atau "API URL" yang tersedia.

    - Api Key : Masukkan Api Key website provider. Contoh : uAgiidsI2bdz1BKUzjcRwxn3

      **Note :**
      Untuk mendapatkan api key provider kamu dapat masuk ke menu "akun" atau profile" pada website provider, kemudian salin "URL Request" atau "API URL" yang tersedia.

    - Tipe : Pilih tipe berupa : 
      Luar -> Jika web provider dari luar negeri
      Indo -> Jika web provider dari indonesia.

4. Jika sudah klik "Simpan".
5. Jika sudah berhasil disimpan maka akan muncul pemberitahuan seperti berikut.
  </Tab>

3. Klik "Tetap Hapus"
