#### Cara membuat Merchant di Tripay

Untuk membuat merchant atau menambah merchant di Tripay, kamu dapat menggunakan cara berikut : 

1. Login tripay menggunakan akun kamu di website tripay.co.id atau klik link : https://tripay.id/login
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1724656987/mt_ke8e0h.png" />
2. Masuk ke menu "Merchant".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1724656986/mt1_nrxnhu.png" />
3. Klik "Tambah Merchant".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1724656984/mt2_luwph7.png" />
4. Isi kelengkapan data sebagai berikut : 
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1724656984/mt3_ekgxhn.png" />
    - Nama Merchant : Isi dengan nama website/aplikasi kamu.
    - Logo : Isi dengan logo website/aplikasi kamu menggunakan format jpg, jpeg, png dengan ukuran 200 x 50px.
    - URL Website : Isi dengan URL atau link website kamu.
    - Deksripsi : Isi detail produk/layanan yang di sediakan di website/aplikasi kamu. Jika URL website kamu memerlukan akses data login, sertakan juga data login di kolom deskripsi.
    - URL Callback : Isi dengan URL Callback yang tertera di kolom Konfigurasi Tripay pada website kamu, atau klik link http://192.168.100.2/endgames/public/sistem/handle/callback/tripay.
        <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1724656983/mt4_uozznc.png" />
    - Whitelist IP : Isi dengan alamat IPv4 dan IPv6 dari server hosting kamu. Jika IP lebih dari 1, pisahkan dengan koma. Contoh: 192.168.1.1,192.168.1.2.
        <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1724656984/mt5_xq5laj.png" />
5. Jika sudah klik "Simpan".
    <img width="30%" src="https://res.cloudinary.com/dvnlxmymx/image/upload/v1724656984/mt6_gq8nfa.png" />
