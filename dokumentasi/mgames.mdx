MGames dilengkapi dengan fitur pembelian Guest, yang dimana pengguna atau user dapat bertransaksi tanpa harus login. MGames juga dilengkapi sistem pembayaran menggunakan saldo akun.
MGames dibuat menggunakan Framework CodeIgniter 4 dan dikembangkan oleh Enivay Company. MGames dilengkapi dengan sistem otomatis 2 Provider Transaksi dan 1 Provider Payment Gateway.'

## Konfigurasi Umum

Kolom Umum pada menu Konfigurasi merupakan menu untuk mengganti Nama Website, Judul Website, Logo Website, Author, Keywords, & Deksripsi.

Berikut langkah untuk melakukan konfigurasi Umum :

1. Klik "Konfigurasi" pada halaman admin.
    <img width="100%" src="/images/checks-passed.png" />
2. Pada kolom "Umum", silahkan edit atau ganti konfigurasi yang diinginkan sebagai berikut :
    - Nama Website : Isi dengan nama website kamu.
    - Judul Website : Isi dengan judul webiste kamu.
    - Logo Website : Pilih atau masukkan logo website kamu.
    - Keywords : Isi sesuai dengan nama website kamu.
    - Deksripsi : Isi dengan deksripsi singkat tentang website kamu.
3. Jika sudah klik "Simpan".

## Konfigurasi Tripay

Konfigurasi Tripay merupakan konfigurasi untuk sistem pembayaran website kamu dengan sistem Tripay.

Berikut langkah untuk Konfigurasi Tripay :

1. Klik "Konfigurasi" pada halaman admin.
2. Scroll kebawah, kemudian pada kolom Konfigurasi Tripay silahkan masukkan "Api Key", "Private Key" & "Kode Merchant" kamu.
3. Jika sudah klik "Simpan".


## Membuat Merchant di Tripay

Untuk membuat merchant atau menambah merchant di Tripay, kamu dapat menggunakan cara berikut :

1. Login tripay menggunakan akun kamu.
2. Masuk ke menu "Merchant".
3. Klik "Tambah Merchant".
4. Isi kelengkapan data sebagai berikut :

    - Nama Merchant : Isi dengan nama merchant atau toko kamu.
    - Logo : Isi dengan logo merchant atau toko
    - URL Website : Isi dengan URL atau link website
    - Deksripsi : Isi dengan penjelasan atau deksripsi singkat tentang bisnis atau toko kamu
    - URL Callback : Isi dengan URL Callback yang tertera di kolom Konfigurasi Tripay, atau klik link https://demo.enivay.com/mgames/admin/konfigurasi
    - Whitelist IP : Isi dengan IPv6 dari server hosting kamu
5. Jika sudah klik "Simpan".

## Kelola Games

Menu Games merupakan menu untuk melihat seluruh data games yang telah terpasang di website kamu. Pada menu Kelola Games kamu dapat melakukan tambah games, edit games, & hapus games.

<Tabs>
  <Tab title="Tambah Games">
1. Klik menu "Games" pada halaman admin.
2. Klik menu "Tambah" pada pojok kanan atas.
3. Isi kolom Tambah Games sebagai berikut :
    - Games : Isi dengan nama games yang ingin kamu tambahkan.
    - Gambar : Pilih atau Drag and Drop gambar games yang ingin ditambahkan.
    - Kategori : Silahkan pilih "Manual Proses", "Produk Lainnya", atau "Top Up Game".
    - Urutan : Isi dengan nomor urut untuk menampilkan games.
    - Deksripsi : Masukkan deksripsi mengenai games yang ingin ditambahkan secara singkat.
    - Sistem Target : Pilih salah satu sistem taget yang sesuai dengan games yang kamu tambahkan.
    - Validasi Player : Jika menggunakan sistem validasi player silahkan pilih "Ya", namun jika tidak menggunakan sistem validasi player maka pilih "Tidak".
    - Kode Validasi : Masukkan kode validasi jika kamu menggunakan sistem validasi player.
4. Jika sudah, klik "Simpan".
  </Tab>
  <Tab title="Edit Games">
1. Klik menu "Games" pada halaman admin.
2. Klik menu "Edit" pada kolom games yang ingin di edit.
3. Silahkan edit data games sebagai berikut :
    - Games : Isi dengan nama games yang ingin kamu tambahakan.
    - Gambar : Pilih atau Drag and Drop gambar games yang ingin ditambahkan.
    - Kategori : Silahkan pilih "Manual Proses", "Produk Lainnya", atau "Top Up Game".
    - Urutan : Isi dengan nomor urut untuk menampilkan games.
    - Deksripsi : Masukkan deksripsi mengenai games yang ingin ditambahkan secara singkat.
    - Sistem Target : Pilih salah satu sistem taget yang sesuai dengan games yang kamu tambahkan.
    - Validasi Player : Jika menggunakan sistem validasi player silahkan pilih "Ya", namun jika tidak menggunakan sistem validasi player maka pilih "Tidak".
    - Kode Validasi : Masukkan kode validasi jika kamu menggunakan sistem validasi player.
    - Status : Pilih "On" jika ingin memunculkan game di website kamu.
4. Jika sudah klik "Simpan".
  </Tab>
  <Tab title="Hapus Games">
1. Klik menu "Games" pada halaman admin.
2. Klik menu "Hapus" pada kolom games yang ingin di hapus.
3. Klik "Tetap Hapus"
  </Tab>
</Tabs>

## Kelola Pesanan

Kelola Pesanan merupakan menu untuk melihat seluruh data pesanandari seluruh member di website kamu. Pada menu Kelola Pesanan kamu dapat melakukan edit data pesanan & menghapus data pesanan .

Berikut langkah untuk Kelola Pesanan :

<Tabs>
  <Tab title="Edit Data Pesanan">
1. Klik menu "Pesanan" pada halaman admin.
2. Klik menu "Edit" pada kolom pembelian yang ingin di edit.
3. Silahkan edit data pembelian sebagai berikut :
    - No Transaksi : (Tidak dapat di edit)
    - Username : Otomatis terisi dengan nama user yang melakukan pemesanan
    - Nomor WhatsApp : Otomatis terisi dengan nomor whatsapp user
    - Produk : Otomatis terisi dengan produk yang dibeli oleh user, namun dapat diisi atau diganti
    - Metode : Otomatis terisi sesuai dengan metode pembayaran yang dipilih user
    - User ID : Isi atau ganti sesuai dengan User ID user
    - Zone/Server ID : Isi atau ganti sesuai dengan Zone/Server ID user
    - Nickname : Otomatis terisi sesuai dengan User Id & Zone/Server Id yang dimasukkan user
   -  Status : Dapat diisi sebagai berikut :
        1. Pending : Pesanan belum dibayar / menunggu pembayaran
        2. Processing : Pesanan dalam proses oleh provider / manual
        3. Success : Pesanan telah berhasil diproses
        4. Canceled : Pesanan gagal diproses
        5. Expired : Pesanan gagal / expired
    - Keterangan : Dapat diisi dengan keterangan mengenai proses pesanan dll.
4. Jika sudah klik "Simpan".
  </Tab>
  <Tab title="Hapus Data Pesanan">
1. Klik menu "Pesanan" pada halaman admin.
2. Klik menu "Hapus" pada kolom pembelian yang ingin di hapus.
3. Klik "Tetap Hapus"
  </Tab>
</Tabs>


## Kelola Topup Saldo

Menu Top Up adalah menu untuk melihat seluruh data topup member di website kamu. Pada menu ini kamu dapat melakukan Edit Topup & Hapus Topup.

Berikut langkah untuk Kelola Topup Saldo :

<Tabs>
  <Tab title="Edit Topup">
1. Klik menu "TopUp" pada halaman admin.
2. Klik menu "Edit" pada data topup yang ingin di edit.
3. Silahkan edit data topup sebagai berikut :
    - "Topup ID" : Tidak dapat di edit
    - Username : Otomatis terisi sesuai dengan username yang melakukan topup
    - Metode : Otomatis terisi sesuai dengan metode yang dipilih user
    - Jumlah : Otomatis terisi sesuai dengan request yang diajukan user. - - Namun jumlah yang tertera juga dapat diubah berupa dikurangi atau ditambah.
    - Status : Silahkan pilih status berupa :
    - Pending : Topup belum dibayar / menunggu pembayaran
    - Success : Topup telah berhasil diproses
    - Canceled : Topup gagal diproses
4. Jika sudah, silahkan klik "Simpan".
