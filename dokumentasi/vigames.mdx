#### Kelola Games

Kelola Games merupakan menu untuk melihat seluruh games yang telah ditambahkan dan ditampilkan ke website kamu. Selain itu kamu juga dapat melakukan tambah atau edit games di menu ini.

<Tabs>
  <Tab title="Tambah Games">
1. Klik menu "Kelola Games" pada halaman admin.
2. Klik menu "Tambah Games".
3. Silahkan isi kolom sebagai berikut : 
    - Url Gambar : Pilih gambar games yang ingin ditambahkan.
    - Nama Game : Isi dengan nama games yang ingin ditambahkan.
    - Slug : Isi dengan nama games dengan huruf kecil. (Contoh : Mobile Legend -> mobile-legend).
    - Kategori : Pilih salah satu kategori yang telah ditambahkan sebelumnya.
    - Deskripsi : Isi dengan deksripsi singkat tentang games yang ditambahkan.
    - Sistem Target : Pilih salah satu sistem target yang telah ditambahkan sebelumnya.
    - Validasi ID : Pilih "Ya" atau "Tidak".
4. Jika sudah klik "Tambah".
  </Tab>
  <Tab title="Edit Games">
1. Klik menu "Kelola Games" pada halaman admin.
2. Klik menu "Edit" pada kolom games yang ingin di edit.
3. Silahkan edit kolom yang ingin di edit.
4. Jika sudah klik "Simpan".
  </Tab>
</Tabs>

#### Kelola Topup

Menu Kelola Top Up adalah menu untuk melihat seluruh data topup member di website kamu. Pada menu ini kamu dapat melakukan Edit Topup & Hapus Topup.

<Tabs>
  <Tab title="Edit Topup">
1. Klik menu "Kelola TopUp" pada halaman admin.
2. Klik menu "Edit" pada data topup yang ingin di edit.
3. Silahkan edit data topup sebagai berikut : 
    - Topup ID : Tidak dapat di edit
    - Email : Otomatis terisi sesuai dengan pengirim topup
    - Metode : Otomatis terisi
    - Jumlah : Otomatis terisi sesuai dengan request yang diajukan member. Namun jumlah yang tertera juga dapat diubah berupa dikurangi atau ditambah.
    - Status : Silahkan pilih status berupa "Pending", "Success", atau "Canceled".
4. Jika sudah, silahkan klik "Simpan".


***Note :**
Jika ingin menerima topup atau menolak topup kamu bisa langsung klik "Terima" atau "Tolak" pada menu data edit seperti berikut : 

1. Pada menu "Kelola Topup" pada halaman admin, silahkan klik "Edit" pada data topup.

2. Jika ingin menerima topup silahkan klik "Terima", dan jika ingin menolak topup silahkan klik "Tolak" seperti berikut : 
  </Tab>
  <Tab title="Hapus Topup">
1. Klik menu "Kelola TopUp" pada halaman admin.
2. Klik menu "Hapus" pada kolom topup yang ingin di hapus.
3. Klik "Tetap Hapus"
  </Tab>
</Tabs>

#### Kelola Pembelian

Kelola Pembelian merupakan menu untuk melihat seluruh data pembelian dari seluruh member di website kamu. Pada menu Kelola Pembelian kamu dapat melakukan edit data pembelian.

Berikut cara Edit Pembelian : 

1. Klik menu "Kelola Pembelian" pada halaman admin.
2. Klik menu "Edit" pada kolom pembelian yang ingin di edit.
3. Silahkan edit kolom status sesuai dengan status pembelian berupa "Pending", "Processing", "Completed", & "Canceled".
    - Pending : Jika pembelian masih dalam antrian menunggu di proses.
    - Processing : Jika pembelian sedang di proses.
    - Completed : Jika pembelian telah berhasil di proses atau telah selesai di proses.
    - Canceled : Jika pembelian tidak berhasil di proses karena beberapa kendala.
4. Jika sudah klik "Simpan".

#### Cek Pesanan

Untuk mengecek daftar pembelian maupun status pesanan kamu, kamu dapat melakukan cara berikut : 

<Tabs>
  <Tab title="Cara 1">
Cara pertama dapat melalui menu utama seperti berikut : 

1. Klik "Status Pembelian" pada halaman member.
2. Silahkan masukan Order ID pesanan kamu pada kolom pencarian.
3. Jika sudah klik "Cari".
  </Tab>
  <Tab title="Cara 2">
Cara kedua dapat melalui tombol profile di atas menu utama seperti berikut : 

1. Klik "Profile" pada halaman member.
2. Klik menu "Pesanan".
  </Tab>
</Tabs>


#### TopUp Saldo

Berikut langkah untuk topup saldo member :  

1. Klik "Profile" pada halaman member.
2. Klik menu "Isi Saldo.
3. Masukkan jumlah isi saldo, pada kolom Isi Saldo seperti berikut.
4. Pilih metode yang ingin kamu gunakan.
5. Jika sudah, klik "Isi Saldo".


#### Ganti Password

Berikut langkah untuk mengganti password : 

1. Klik "Profile" pada halaman member.
2. Klik menu "Pengaturan".
3. Isi kolom "Password Lama" kamu, "Password Baru", & "Ulangi Password Baru".
4. Jika sudah klik "Simpan".
