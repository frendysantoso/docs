#### Login Admin

Untuk login sebagai admin dapat dilakukan dengan cara berikut : 

1. Pada tampilan awal atau dashboard silahkan klik "Masuk".
2. Masukkan Email & Kata Sandi.
3. Jika sudah klik "Masuk".


#### Konfigurasi Umum

Tab Umum pada menu Konfigurasi merupakan menu untuk mengganti Nama Website, Icon Website, Logo Website, Keywords, & Deksripsi. Berikut langkah untuk melakukan Konfigurasi Umum : 

1. Klik "Konfigurasi" pada menu administrator di halaman Akun Saya.
2. Klik tab Umum.
3. Silahkan edit atau ganti konfigurasi yang diinginkan sebagai berikut : 
    - Nama Website : Isi dengan Nama Website yang kamu inginkan
    - Icon Website : Pilih atau isi dengan icon website kamu dengan ukuran 512 x 512 px
    - Logo Website : Pilih atau isi dengan logo website dengan ukuran 2560 x 366 px
    - Keywords : Isi dengan keywords yang sesuai dengan website kamu
    - Deksripsi : Isi dengan deksripsi singkat website kamu.
4. Jika sudah klik "Simpan".


#### Konfigurasi Tripay

Tab Tripay pada menu Konfigurasi merupakan menu untuk melakukan konfigurasi pembayaran website kamu dengan sistem tripay. Berikut langkah untuk Konfigurasi Tripay : 

1. Klik "Konfigurasi" pada menu administrator di halaman Akun Saya.
2. Klik tab "Tripay".
3. Silahkan masukkan Api Key, Private Key, & Kode Merchant toko kamu ke kolom "Api Key", Private Key", & "Kode Merchant". 
4. Jika sudah klik "Simpan".

***Note :**
Untuk mendapatkan api key, private key, dan kode merchant kamu bisa klik "Edit" pada merchant kamu. Kemudian salin "Api Key", Private Key", & "Kode Merchant" berikut.


#### Cara membuat Merchant di Tripay

Untuk membuat merchant atau menambah merchant di Tripay, kamu dapat menggunakan cara berikut : 

1. Login tripay menggunakan akun kamu.
2. Masuk ke menu "Merchant".
3. Klik "Tambah Merchant".
4. Isi kelengkapan data sebagai berikut : 
    - Nama Merchant : Isi dengan nama merchant atau toko kamu.
    - Logo : Isi dengan logo merchant atau toko
    - URL Website : Isi dengan URL atau link website
    - Deksripsi : Isi dengan penjelasan atau deksripsi singkat tentang bisnis atau toko kamu
    - URL Callback : Isi dengan URL Callback yang tertera di kolom Konfigurasi Tripay, atau klik link https://demo.enivay.com/valorant/sistem/vid/callback/tripay dibagian tab Tripay 
    - Whitelist IP : Isi dengan IPv6 dari server hosting kamu
5. Jika sudah klik "Simpan".


#### Kelola Pengguna

Kelola Pengguna merupakan menu untuk melihat seluruh data pengguna atau member yang telah mendaftar di website kamu. 

Pada menu Kelola Pengguna kamu dapat melakukan edit data pengguna seperti, mengubah status ataupun mengubah level member.

Berikut cara untuk Edit Pengguna : 

1. Klik menu "Pengguna" pada menu administrator di halaman Akun Saya.
2. Klik menu "Edit" pada kolom pengguna yang ingin di edit.
3. Silahkan edit data pengguna sebagai berikut : 
    - Nama Lengkap : Nama akun pengguna/member
    - Email : Email pengguna/member (Tidak dapat di ubah/edit!!)
    - No. Whatsapp : Nomor whatsapp pengguna/member
    - Status : Pilih status berupa "On" atau "Off"
    - Level : Pilih level berupa "Member" atau "Admin"
4. Jika sudah klik "Simpan".


#### Kelola Transaksi

Menu Transaksi merupakan menu untuk menampilkan seluruh data transaksi pembelian dari seluruh member di website kamu. Pada menu Transaksi kamu dapat melakukan edit data pembelian.

Berikut langkah untuk Edit Pembelian : 

1. Klik menu "Transaksi" pada menu administrator di halaman Akun Saya.
2. Klik menu "Edit" pada kolom pembelian yang ingin di edit.
3. Silahkan edit kolom status sesuai dengan status pembelian berupa "Pending", "Processing", "Completed", & "Canceled".
    - Email : edit atau isi dengan email pengguna yang melakukan pembelian
    - No. Invoice : Otomatis terisi dan tidak dapat di edit!
    - No. Whatsapp : edit atau isi dengan nomor whatsapp pengguna yang melakukan pembelian
    - Status dapat diisi sebagai berikut : 
    - Pending : Jika pembelian masih dalam antrian untuk di proses
    - Processing : Jika pembelian sedang di proses
    - Completed : Jika pembelian telah berhasil di proses atau telah selesai di proses
    - Canceled : Jika pembelian tidak berhasil di proses karena mengalami kendala
4. Jika sudah klik "Simpan".

#### Kelola Metode

Kelola Metode merupakan menu untuk menampilkan seluruh data metode pembayaran yang telah ditambahkan di website kamu. Selain itu menu Metode juga merupakan menu untuk melakukan tambah, edit, & hapus metode. 

Berikut cara untuk kelola metode : 

<Tabs>
  <Tab title="Tambah Metode">
1. Klik menu "Metode" pada menu administrator di halaman Akun Saya.

3. Silahkan isi kolom sebagai berikut :

      *Jika menggunakan provider Manual, silahkan isi dengan No. Rekengin & Atas Nama.

    - Status : Pilih On jika ingin menampilkan metode di website kamu. Pilih Off jika ingin mematikan atau tidak ingin menampilkan metode.

5. Jika metode telah berhasil ditambahkan maka akan muncul pemberitahuan seperti berikut.

#### Cara Transaksi

Berikut langkah untuk transaksi guest : 

1. Pada halaman utama silahkan isi data sebagai berikut : 
    - Pilih Rank Awal : silahkan pilih Rank Awal sesuai dengan Rank di akun Valorant mu.
    - Pilih Rank Tujuan : silahkan pilih Rank Tujuan sesuai dengan yang kamu mau.
    - Pilih RR Awal : silahkan pilih RR Awal sesuai dengan RR di akun Valorant mu.
    - Pilih Server : silahkan pilih Server sesuai dengan Server di akun Valorant mu.
    - Pilih Metode Pembayaran : silahkan pilih metode pembayaran.
    - Pilih TV Proses : silahkan pilih waktu proses yang kamu inginkan.
    - Pilih TV Agent : silahkan pilih agent yang kamu inginkan.
2. Jika sudah silahkan klik "Checkout".
3. Setelah itu silahkan lengkapi data pada popup seperti berikut.
    - Email Address : Silahkan isi dengan email yang masih aktif.
    - Password : Masukkan Password akun kamu.
    - No.WhatsApp : Isi dengan nomor whatsapp kamu.
4. Jika sudah maka akan muncul rincian seperti berikut : 
5. Selanjutnya kamu hanya perlu transfer sesuai dengan jumlah yang tertera. Jika sudah kamu hanya perlu menunggu konfirmasi selanjutnya.
